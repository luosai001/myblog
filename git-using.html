官方网站git学习教程： http://git-scm.com/book/zh/v1

常用命令： 1.获取远程代码 git clone git://github.com/schacon/ticgit.git 2.添加远程仓库 git remote add pb git://github.com/paulboone/ticgit.git 3.获取远程仓库有的，本地仓库没有的 git fetch pb 4.从远程仓库抓取数据 git fetch [remote-name] 5.推送数据到远程仓库 git push origin master 6.查看远程仓库信息 git remote show origin 7.远程仓库的删除和重命名 git remote rename pb paul git remote rm paul 8.显示分支 git branch 9.查看本地、远程仓库分支 git branch -a 10.合并分支 git merge 11.删除分支 git branch -d testing 12.git fetch：相当于是从远程获取最新版本到本地，不会自动merge git fetch origin master git log -p master origin/master git merge origin/master 首先从远程的origin的master主分支下载最新的版本到origin/master分支上 然后比较本地的master分支和origin/master分支的差别 最后进行合并 13 远程仓库跟新到本地，并合并。 git pull origin master
